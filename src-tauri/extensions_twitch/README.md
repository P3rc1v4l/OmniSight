# Twitch-Erweiterungen (nur Windows)

Dieser Ordner wird **nur** für Twitch-Streams als WebView2-Erweiterungs-Pfad genutzt.
Inhalt (vom CI-Build befüllt, nicht im Repo):

- `betterttv/` – BetterTTV (Twitch-Verbesserungen), nur bei Twitch geladen.
- `uBlock0.chromium/` – Kopie von uBlock, damit auch auf Twitch Werbung blockiert wird.

Der Release-Workflow lädt BetterTTV und kopiert uBlock hierher. Lokal ggf. manuell ablegen.
