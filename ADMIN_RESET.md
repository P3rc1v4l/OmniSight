# OmniSight – Admin-Tastenkombination (NICHT in Git hochladen!)

## PIN-Reset für gesperrte Profile

Falls ein Profil mit einem PIN gesperrt ist und der PIN vergessen wurde,
kann dieser nur über folgende Tastenkombination zurückgesetzt werden:

**Kombination:** `Strg + Shift + Alt + R`

Diese Kombination öffnet ein verstecktes Admin-Fenster, in dem du ein
gesperrtes Profil ohne PIN zurücksetzen kannst.

> ⚠️ Diese Datei ist in `.gitignore` eingetragen und wird NICHT auf GitHub hochgeladen.
> Bewahre diese Information sicher auf.
